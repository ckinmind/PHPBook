### 一、简单文件系统操作

####  1.查看磁盘和目录的容量
```
# 使用 df 命令查看磁盘的容量
df
# 人性化显示大小
df -h

# 使用 du 命令查看目录的容量
du
# 人性化显示大小
du -h
# -d参数指定查看目录的深度
du -h -d 0    # 只查看1级目录的信息
du -h -d 1 ~  # 查看2级

```
